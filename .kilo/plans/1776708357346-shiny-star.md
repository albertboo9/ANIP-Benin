# Plan: Enhance Main Titles with Gradient and Larger Size

## Overview
Enhance the main section titles by adding the same gradient color used in the main title "PGI de Pilotage..." and slightly increasing their font size.

## Current State Analysis
**Current Title Styling:**
- Section titles: `clamp(2rem, 4.5vw, 2.75rem)` font size
- Color: `var(--color-text-primary)` (solid color)
- Uppercase, centered, with blue underline

**Target Gradient:**
- The main title uses: `linear-gradient(135deg, var(--color-primary) 0%, var(--color-benin-green) 100%)`
- Applied with `background-clip: text` and `text-fill-color: transparent`

**User Requirements:**
- Apply the same gradient to main section titles
- Slightly increase font size
- Maintain uppercase, centered layout and blue underline

## Objectives
1. **Apply Gradient Color**: Use the same blue-to-green gradient from the main title
2. **Increase Font Size**: Slightly larger than current `clamp(2rem, 4.5vw, 2.75rem)`
3. **Maintain Design Consistency**: Keep all other styling (uppercase, centering, underline)
